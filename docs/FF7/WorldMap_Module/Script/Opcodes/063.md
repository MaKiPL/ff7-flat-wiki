- Opcode: **0x063**
- Name: push greaterequal
- Two-word opcode: No

#### Parameters

| Location | Name |   Description    |
|:--------:|:----:|:----------------:|
|  Stack   |  A   | Value to compare |
|  Stack   |  B   | Value to compare |

#### Notes

Pushes A \>= B to the stack.
