title: 313
---

- Opcode: **0x313**
- Name: set point terrain BGR
- Two-word opcode: No

#### Parameters

| Location | Name | Description |
|:--------:|:----:|:-----------:|
|  Stack   |  B   |  Blue tint  |
|  Stack   |  G   | Green tint  |
|  Stack   |  R   |  Red tint   |

#### Notes

Light effect to be applied to the terrain affected by this light source.
