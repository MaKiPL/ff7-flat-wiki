- Opcode: **0x307**
- Name: set control lock
- Two-word opcode: No

#### Parameters

| Location |     Name     |  Description  |
|:--------:|:------------:|:-------------:|
|  Stack   | Control lock | Boolean value |

#### Notes

0 - disable controls, 1 - enable controls

Enabling controls possibly restores the camera? Needs verification.
