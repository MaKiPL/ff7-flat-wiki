---
title: 34b
---

- Opcode: **0x34b**
- Name: set chocobo type
- Two-word opcode: No

#### Parameters

| Location | Name | Description  |
|:--------:|:----:|:------------:|
|  Stack   | Type | Chocobo type |

#### Notes

Sets the color and walkmesh parameters of the chocobo.

`0 - yellow chocobo`  
`1 - green chocobo`  
`2 - blue chocobo`  
`3 - black chocobo`  
`4 - gold chocobo`
