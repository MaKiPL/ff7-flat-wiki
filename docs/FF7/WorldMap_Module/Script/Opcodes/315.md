---
title: 315
---

- Opcode: **0x315**
- Name: set point sky top BGR
- Two-word opcode: No

#### Parameters

| Location | Name | Description |
|:--------:|:----:|:-----------:|
|  Stack   |  B   |  Blue tint  |
|  Stack   |  G   | Green tint  |
|  Stack   |  R   |  Red tint   |

#### Notes

Light effect to be applied to the top portion of sky in the area affected by this light source.
