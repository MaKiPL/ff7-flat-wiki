---
title: 30b
---

- Opcode: **0x30b**
- Name: set active entity y offset
- Two-word opcode: No

#### Parameters

| Location | Name |          Description           |
|:--------:|:----:|:------------------------------:|
|  Stack   |  Y   | Y offset to apply to the model |

#### Notes

Applies an offset to the Y position of the model associated with the current active entity.
