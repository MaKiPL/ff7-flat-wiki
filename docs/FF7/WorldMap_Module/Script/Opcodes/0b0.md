title: 0b0
---

- Opcode: **0x0b0**
- Name: push logicand
- Two-word opcode: No

#### Parameters

| Location | Name |  Description  |
|:--------:|:----:|:-------------:|
|  Stack   |  A   | Boolean value |
|  Stack   |  B   | Boolean value |

#### Notes

Pushes 1 to the stack if both A and B are not 0, otherwise 0.
