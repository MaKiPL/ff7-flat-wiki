---
title: 01b
---

- Opcode: **0x01b**
- Name: push direction from active entity to point
- Two-word opcode: No

#### Parameters

| Location |   Name   |          Description          |
|:--------:|:--------:|:-----------------------------:|
|  Stack   | Point ID | Point to compare direction to |

#### Notes

Pushes the direction towards the point specified from the active entity
