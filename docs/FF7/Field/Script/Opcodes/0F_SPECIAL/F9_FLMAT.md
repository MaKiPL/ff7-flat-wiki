---
title: F9_FLMAT
---

- Opcode: **0x0FF9**
- Short name: **SPECIAL: FLMAT**
- Long name: Special: Fill Materia

#### Memory layout

| 0x0F | 0xF9 |
|------|------|

#### Arguments

None.

#### Description

Fills all available materia entries so that the party has the maximum amount of materia that can be carried.
